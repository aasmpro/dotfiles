    Running docs

    Installation#

    we use mkdocs as main platform: 

    1
    pip install mkdocs

    and mkdocs-material as theme: 

    1
    pip install mkdocs-material

    Running#

    1
    mkdocs serve

    1
    mkdocs serve -a localhost:8001

    Building#

    1
    mkdocs build --clean
